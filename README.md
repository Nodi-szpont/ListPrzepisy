# Lista Przepisów

Aplikacja do zarządzania listą przepisów kulinarnych z kategoryzacją.

## Nauczasz się:
- **ListView** - wyświetlanie kategorii i przepisów
- **Intent** - przechodzenie między aktywności
- **ArrayAdapter** - obsługa danych listy
- **AdapterView.OnItemClickListener** - obsługa wyboru elementu
- **String.putExtra()** - przekazywanie danych między aktywności
- **Toast** - wyświetlanie komunikatów
- **Getter/Setter** - organizacja danych przepisu
- **Modele danych** - klasa Przepis
- **Color** - zmiana kolorów
